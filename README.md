# France Top 50 Playlist Analysis

## 📊 Project Overview
This project analyzes audience sensitivity, content compliance, and format preference in the France Top 50 playlist.

## 🚀 Features
- Explicit vs Clean Analysis
- Album Format Distribution
- Song Duration Insights
- Rank-tier Comparison
- Interactive Streamlit Dashboard

## 🛠 Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib

## ▶️ Run Locally
streamlit run app.py
