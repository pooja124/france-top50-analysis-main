# France Top 50 Playlist Analysis

## 📊 Project Overview
This project analyzes audience sensitivity, content compliance, and format preference in the France Top 50 playlist.

## 🚀 Features
- Explicit vs Clean Analysis
- Album Format Distribution
- Song Duration Insights
- Rank-tier Comparison
- Interactive Streamlit Dashboard

## 🛠 Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib

## 🚀 How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/pooja124/france-top50-analysis-main.git
cd france-top50-analysis-main
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Streamlit App

```bash
streamlit run app.py
```
